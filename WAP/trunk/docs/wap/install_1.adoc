## 环境要求

NOTE:  运行环境要求和开发环境推荐

### 构建工具
**Maven3**

* 下载地址： http://maven.apache.org/download.cgi
* 配置环境变量
** path：增加maven的bin目录，可以识别  `mvn` 命令
** MAVEN_HOME：指定maven目录

### Java运行环境
**JDK1.7**

* 下载地址： https://jdk7.java.net/download.html
* 配置环境变量
** path：增加jdk的bin目录，可以识别 `java` 命令
** JAVA_HOME：指定java目录

### 开发IDE推荐
**IDEA13**

* 下载地址： http://www.jetbrains.com/idea/download/
