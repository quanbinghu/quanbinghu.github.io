ASKIT

ASKIT （Application Service Kit）
支持开发基于RESTFul协议服务接口的开发套件 ！

ASKIT后端应用服务框架，基于JavaEE界主流技术框架的封装和再开发而提供了更为简洁的使用方法和更加强大的基础功能。 运行环境需要支持JDK7和Servlet3，构建工具采用Maven；

SPRING    JPA    SHIRO    RESTFUL

AUIKIT

AUIKIT(Adminstrator UI Kit)
解决管理者类型项目的前端开发集成工具包，源于框架高于框架！

AUIKIT管理者类型项目前端开发工具包，基于前端开发界主流框架的封装，对界面的控制、展现都进行了精心研发和调整。 并且使用目前主流的开发工具和开发方式，使得调用、扩展和再次开发都非常容易、灵活；

BOOTSTRAP3    HTML5    CSS3    ANGULARJS

APPDOCK

应用运行（PaaS）平台
支持多语言、多中间件可弹性配置的应用运行平台！

APPDOCK是一个应用运行引擎，核心采用Docker开源项目，基于Linux Container的资源独享型PaaS。并对其进行了可管理化开发，提供多中间件、多语言、弹性的服务端运行环境，能够快速的部署应用的PaaS平台；

LINUX    DOCKER    COREOS
